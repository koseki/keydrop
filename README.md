# Keydrop

## Deploying

Install the [Heroku Toolbelt](https://toolbelt.heroku.com/).

```sh
$ git clone git@github.com:koseki/keydrop.git # or clone your own fork
$ cd keydrop
$ heroku create
$ git push heroku master
$ heroku open
```
